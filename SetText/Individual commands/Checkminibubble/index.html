    Checkminibubble¶

    Redirect if any minibubble are present to a different dialogue line or do nothing if none are present.

    Syntax¶

    |checkminibubble,lineid|

    Parameters¶

    lineid: int¶

    The Dialogue line id to redirect if applicable. This must be a valid Dialogue line id or an exception will be thrown.

    Remarks¶

    This command was specifically made to allow the Gen & Eri switch puzzle to work inside the Honey Factory.
